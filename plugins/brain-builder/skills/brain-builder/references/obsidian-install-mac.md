# Obsidian Install — Mac

## The 60-second install

1. Go to https://obsidian.md
2. Click "Get Obsidian for Mac"
3. The download is a .dmg file (~100MB)
4. Open the .dmg file
5. Drag the Obsidian icon to the Applications folder
6. Open Applications → Obsidian
7. macOS may say "Apple cannot verify..." — click Cancel, then right-click Obsidian.app → Open → Open. After the first open, it works normally.
8. Skip the Obsidian welcome wizard ("Quick start" / "Sync"). Click "Open folder as vault" instead.

## Linux note

On Linux: download the AppImage from obsidian.md, make it executable (`chmod +x Obsidian-*.AppImage`), and run it. Or use Snap: `sudo snap install obsidian --classic`.

## Sanity check

Open Obsidian. You should see a left sidebar with "File explorer." If you do, install is complete.
