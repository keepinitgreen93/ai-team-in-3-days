# Obsidian Install — Windows

## The 60-second install

1. Go to https://obsidian.md
2. Click "Get Obsidian for Windows"
3. The download is a .exe installer (~100MB)
4. Run the installer. Accept defaults.
5. Windows Defender may flag it — click "More info" → "Run anyway". Obsidian is signed by the Obsidian team and is safe.
6. Open Obsidian from the Start Menu
7. Skip the Obsidian welcome wizard. Click "Open folder as vault" instead.

## Sanity check

Open Obsidian. You should see a left sidebar with "File explorer." If you do, install is complete.

## PowerShell note

If the user prefers winget: `winget install Obsidian.Obsidian`
