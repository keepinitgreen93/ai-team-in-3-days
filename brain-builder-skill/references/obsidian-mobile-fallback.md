# Mobile / No-Computer Fallback

The Brain is just markdown files. It doesn't HAVE to be Obsidian.

## Mobile-only options

1. **Obsidian mobile** — free on iOS and Android. Same UI as desktop. Best option for mobile-only users.
2. **iA Writer** — beautiful markdown editor for iOS/Mac
3. **Markor** — Android markdown editor
4. **Plain text editor** — any notes app that supports `.md` files (Bear, Notion can export, Standard Notes)

## What changes if not Obsidian

- Wiki-links (`[[page]]`) won't auto-render but they STILL WORK as plain text — you can search for them
- The Brain folder structure still lives on disk; sync via iCloud/Google Drive/Dropbox
- AI Employees still read the markdown files directly — they don't care which editor created them

## Recommended for mobile-only

Install Obsidian mobile. Sync your Brain folder via iCloud (iOS) or Google Drive (Android). It's free and works the same as desktop.
