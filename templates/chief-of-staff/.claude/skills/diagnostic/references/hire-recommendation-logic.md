# Hire Recommendation Logic

After the diagnostic, recommend the user's TOP 2 highest-leverage AI Employees. Use the patterns below to map answers to role types. The role recommendations should reference specific Q&A answers.

## Diagnostic-to-role patterns

| Pattern in answers (Q7-Q11) | Recommended role |
|---|---|
| "I'm spending hours on content / blog / writing / posts / emails / newsletters" | **Content Designer** |
| "I'm losing time on social media / posting / Instagram / TikTok / engagement" | **Social Media Manager** |
| "My website looks bad / no email signup / no landing pages / want to redesign" | **Web Designer** |
| "Calendar / inbox / scheduling / files / admin work / organization" | **Admin Assistant** |
| "Comparing prices / vendor research / estimates / quotes" | **Research & Estimate Specialist** |
| "I want to launch a workshop / course / training / webinar" | **Workshop Builder** |
| "Editing videos / producing video / YouTube / podcasts" | **Video Producer** |
| "Lead research / prospecting / qualification / sales outreach" | **Lead Researcher** |
| "Customer service / DMs / response time / replying to messages" | **Customer Service Responder** |
| "Bookkeeping / categorizing transactions / invoicing / financial reports" | **Bookkeeping Helper** |
| "Reviews / testimonials / online reputation" | **Reputation Manager** |
| "Email sequences / drip campaigns / lifecycle / welcome series" | **Email Specialist** |
| "Tools for clients / apps / quizzes / trackers / habit logs" | **App Designer** |
| "SEO / search ranking / Google / organic traffic" | **SEO Specialist** |

## Pick rules

- Look at Q11 ("which would change the business the MOST?") — that pattern wins #1
- Pick the SECOND-highest pattern from Q7-Q10 for #2
- If the user only mentioned one area, pick a role that COMPLEMENTS the first one (Content Designer + Social Media Manager pair well; Web Designer + Content Designer pair well; etc.)

## Compounding effects

When picking #2, prefer roles whose output COMPOUNDS with #1's:

- Content Designer (recipes/posts) → Social Media Manager (turns them into platform-native posts) → Web Designer (uses them as featured content on landing pages)
- Web Designer (homepage refresh) → Content Designer (writes the new copy)
- Lead Researcher (finds prospects) → Email Specialist (writes the outreach sequence)

## What to write in the memo

For each recommended role, include:

1. **Why this fits** — quote the specific diagnostic answer that pointed here ("You said you spend 10 hours a week trying to keep up with social posts...")
2. **What they'll produce in their first week** — 3-5 concrete deliverables specific to the user's business (not generic). Example for a wellness coach + Content Designer: "5 fresh recipes in your voice, a wellness guide on stress management, a Mother's Day promo email."
3. **Why this one first** — order rationale based on Q11 (highest impact first)

## When uncertain

If the diagnostic answers are too vague to recommend confidently, write:

> "I'd love to recommend a hire, but I need a little more context. Tell me more about [specific area that was vague]."

Never guess. Real recommendations only.
